Distinguishes the direct-greta entry from the asreml-style / brms-style
formula entries by listing target parameters and the inferred likelihood
family.

``` r
# S3 method for class 'flexybayes_direct_greta'
print(x, ...)
```

## Arguments

- x:

  A `flexybayes_direct_greta` object.

- ...:

  Additional arguments (ignored).

## Value

Invisibly returns `x`.
