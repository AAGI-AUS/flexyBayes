# Fixed-effect model formula of a per-row INLA fit

The fixed-effect (population-level) formula, recovered from the captured
call. Random-effect and residual-structure terms are not part of this
formula.

## Usage

``` r
# S3 method for class 'flexybayes_inla'
formula(x, ...)
```

## Arguments

- x:

  A `flexybayes_inla` fit.

- ...:

  Ignored.

## Value

A `formula`.
