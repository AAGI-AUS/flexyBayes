# marginaleffects support: covariance (greta backend)

marginaleffects support: covariance (greta backend)

## Usage

``` r
# S3 method for class 'flexybayes'
get_vcov(model, ...)

# S3 method for class 'flexybayes_inla'
get_vcov(model, ...)
```

## Arguments

- model:

  A `flexybayes_inla` fit.

- ...:

  Ignored.

## Value

Fixed-effect covariance matrix.
